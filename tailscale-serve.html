Install Nextcloud AIO
with Tailscale Serve

A simple, step-by-step guide to self-hosting Nextcloud All-in-One on Docker Desktop, privately accessible via Tailscale.

⚡ Before You Start

  • Docker Desktop installed
  • Tailscale account + app
  • Windows, macOS or Linux
  • ~10 GB free disk space
  • Terminal / PowerShell
01
Install & Start Docker Desktop
docker.com/products/docker-desktop
›
Download Docker Desktop for your OS and make sure it's running before continuing. You should see the Docker whale icon in your system tray.
📥
Download & Install
Go to docker.com, download Docker Desktop for your platform, and run the installer. Accept the default settings.
▶️
Start Docker & verify
Open Docker Desktop. Once it shows "Engine running", verify in terminal:
docker version
ℹ️ On Windows, make sure WSL 2 backend is enabled in Docker Desktop → Settings → General.
02
Install & Connect Tailscale
tailscale.com/download
›
Install the Tailscale app and log in. Your machine will get a stable private IP (100.x.x.x) and a MagicDNS name like mypc.tail1234.ts.net.
🔑
Install & Log in
Download and install Tailscale, then sign in with your account. Your machine joins your tailnet automatically.
📋
Note your Tailscale hostname
Find your MagicDNS hostname — you'll need it later. Run:
tailscale status
Look for your machine name, e.g. mypc.tail1234.ts.net
ℹ️ Make sure MagicDNS is enabled in your Tailscale admin console at login.tailscale.com → DNS.
03
Create a Docker Volume for Nextcloud
persistent storage setup
›
Nextcloud AIO needs a named Docker volume to store all your data persistently. This survives container restarts.
💾
Create the volume
Run this once in your terminal:
docker volume create nextcloud_aio_mastercontainer
04
Run Nextcloud AIO Container
the main install command
›
This single command pulls and starts the Nextcloud AIO master container. It exposes the AIO management interface on port 8080.
🐳
Run the container
Paste this into your terminal (one command):
docker run \ --init \ --sig-proxy=false \ --name nextcloud-aio-mastercontainer \ --restart always \ --publish 8080:8080 \ --env APACHE_PORT=11000 \ --env APACHE_IP_BINDING=127.0.0.1 \ --volume /var/run/docker.sock:/var/run/docker.sock:ro \ --volume nextcloud_aio_mastercontainer:/mnt/docker-aio-config \ -d nextcloud/all-in-one:latest
⚠️ On Windows, replace /var/run/docker.sock with //var/run/docker.sock or use the named pipe: \\.\pipe\docker_engine:/var/run/docker.sock
ℹ️ APACHE_IP_BINDING=127.0.0.1 ensures Nextcloud only binds locally — Tailscale Serve will handle the public exposure.
05
Set Up Nextcloud via AIO Interface
initial configuration wizard
›
The AIO management portal walks you through starting all Nextcloud services. Open it in your browser and follow the wizard.
🌐
Open AIO Management Portal
In your browser, navigate to:
https://localhost:8080
Accept the self-signed cert warning (click Advanced → Proceed).
🔐
Get the initial password
The page shows a one-time passphrase. Copy it. You'll log in with it to access the AIO dashboard.
⚙️
Enter your domain & start containers
When asked for a domain, enter your Tailscale hostname (from Step 2), e.g.:
mypc.tail1234.ts.net
Then click "Start containers" to spin up Nextcloud, the database, Redis, and other services.
⚠️ Do NOT enter a domain with https:// — just the bare hostname like mypc.tail1234.ts.net.
06
Configure Tailscale Serve
secure HTTPS proxy via tailnet
›
Tailscale Serve creates an HTTPS proxy to your local Nextcloud port, with automatic TLS certificates — no port forwarding or domain setup needed.
🔧
Enable Tailscale Serve for Nextcloud
Nextcloud AIO's Apache runs on port 11000 locally. Tell Tailscale to serve it over HTTPS:
tailscale serve https / http://localhost:11000
✅
Verify the serve configuration
Confirm it's active:
tailscale serve status
You should see your hostname with HTTPS forwarding to http://localhost:11000.
ℹ️ Tailscale Serve only exposes the URL to your tailnet (devices logged in to your Tailscale account). Use tailscale funnel instead if you want public internet access.
07
Access & Finish Setup
you're almost there!
›
With Tailscale Serve running and Nextcloud containers started, you can now access your private cloud from any device on your tailnet.
🚀
Open your Nextcloud
From any device connected to your Tailscale network, visit:
https://mypc.tail1234.ts.net
Replace with your actual Tailscale hostname from Step 2.
👤
Create your admin account
The Nextcloud setup wizard will ask you to create an admin username and password. Store these safely — this is your main account.
📱
Connect mobile & desktop apps
Install the Nextcloud app on your phone or desktop. When prompted for server URL, enter your Tailscale hostname. Make sure Tailscale is running on that device too.
🎉 That's it! Your private Nextcloud is running, secured by Tailscale's encrypted network with automatic HTTPS.

🎉 Your Cloud is Ready

Access Nextcloud from any device on your Tailscale network. All traffic is end-to-end encrypted — no ports exposed to the internet.
